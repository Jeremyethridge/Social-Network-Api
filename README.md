# Social-Network-Api

## Description
This is a NoSQL application built using MongoDB. This is a social-network api takes in Users and Thoughts.

## installation
1. Git clone
2. Install packages
3. Node server.js
4. Open testing environment (Insomnia)
5. Test routes
